Detailed Description

Size function for the vertices in a graph representing a P2P network. The size of the shapes representing the nodes are given in the constructor. This class has three attributes, that indicate the size of vertices representing peers, the size of vertices representing documents stored by the peers, and a parameter that indicates the limit between integers (vertex labels) representing documents and integers representing peers
