To use the program run the jar file as an executable. Once loaded a sample log file has been provided, in the program go to file>load and select the log file you wish to open.

Use the play speed buttons at the bottom of the program to ajust the speed which the events play. For more detailed information see:
	https://sites.google.com/a/nmai.ca/home/research-projects/p2p-simulation-project/simulation-visualizer
